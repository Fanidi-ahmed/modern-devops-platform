# Troubleshooting checklist

## Avant de conclure qu'un outil "ne marche pas"

- suis-je dans le bon dossier ?
- ai-je créé le fichier au bon endroit ?
- le nom du fichier est-il exact ?
- ai-je bien sauvegardé le fichier ?
- ai-je une faute dans une commande ?
- suis-je dans le bon environnement Python ?
- Git ignore-t-il un fichier que je pensais voir ?
- ai-je relu le message d'erreur exact ?

## Bonnes habitudes

- copier le message d'erreur exact
- noter la commande exécutée
- tester une seule chose à la fois
- ne pas corriger au hasard
- comprendre la cause avant de patcher
